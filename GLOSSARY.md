# GLOSSARY.md — Terminologie Licornes

> Vocabulaire magique du Royaume des Licornes.

---

## Personnages

| Terme | Description |
|-------|-------------|
| **Andrea** | Profil pour les 3 ans (icône : petite licorne rose) |
| **Suzanne** | Profil pour les 6 ans (icône : grande licorne violette) |
| **Licorne** | Cheval magique avec une corne sur le front |
| **Pégase** | Licorne avec des ailes (bonus) |

---

## Lieux du Royaume

| Terme | Description |
|-------|-------------|
| **Royaume des Licornes** | Le monde magique du site |
| **Château Arc-en-ciel** | Lieu principal, accueil |
| **Forêt Enchantée** | Zone de jeux |
| **Cascade Scintillante** | Zone de coloriage |
| **Prairie Étoilée** | Zone des histoires |
| **Grotte Mystérieuse** | Zone des surprises |

---

## Actions

| Terme | Description |
|-------|-------------|
| **Colorier** | Activité de coloriage tactile |
| **Jouer** | Accéder aux mini-jeux |
| **Écouter** | Activer sons ou comptines |
| **Créer** | Personnaliser sa licorne |
| **Explorer** | Naviguer sur la carte |
| **Découvrir** | Ouvrir une surprise |

---

## Éléments magiques

| Terme | Description |
|-------|-------------|
| **Étoile** | Récompense / point |
| **Arc-en-ciel** | Élément décoratif principal |
| **Paillettes** | Animation de particules |
| **Magie** | Effet sonore et visuel |
| **Nuage** | Fond et transitions |

---

## Récompenses

| Terme | Description |
|-------|-------------|
| **Étoile dorée** | Jeu terminé avec succès |
| **Couronne** | Badge spécial |
| **Ailes** | Déblocage bonus |
| **Cœur** | Favori sauvegardé |
